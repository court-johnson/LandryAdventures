== Description ==

grishma is a very clean and simple blog theme. It features a clean dynamic grid layout, and the ability to customize right from the customization menu. Of course it is fully responsive, supports custom menus and widgets and comes with an social profile link feature. The theme is best for personal blog and magazine websites. Theme Demo here http://demos.themingpress.com/grishma/.

== Installation ==

1. Sign into your WordPress dashboard, go to Appearance > Themes, and click Add New.
2. Click Add New.
3. Click Upload.
4. Click Choose File and select the theme zip file you downloaded from WordPress.org.
5. Click Install Now.
6. Click Add New, then click Upload, then click Choose File.
7. After WordPress installs the theme, click Activate.
8. You've successfully installed your new theme!
